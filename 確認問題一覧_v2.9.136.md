# 確認問題一覧 2.9.136

| 用語 | 問題 | 正解 |
|---|---|---|
| 感染源 | 感染源の説明として正しいものはどれか。 | 病原体を持ち、感染のもとになる人・動物・環境・物品 |
| 感染経路 | 感染経路の説明として正しいものはどれか。 | 病原体が感染源から宿主へ移る道筋 |
| 消毒用エタノール | 血液が付着した器具を消毒用エタノールで消毒する条件はどれか。 | 76.9～81.4容量％に10分間以上浸す |
| 次亜塩素酸ナトリウム | 血液が付着した器具に用いる次亜塩素酸ナトリウムの条件はどれか。 | 0.1％液に10分間以上浸す |
| グルコン酸クロルヘキシジン | グルコン酸クロルヘキシジンによる器具消毒の条件はどれか。 | 0.05％液に10分間以上浸す |
| 塩化ベンザルコニウム | 逆性石けんによる器具消毒の条件はどれか。 | 0.1～0.2％液に10分間以上浸す |
| 滅菌 | 滅菌の説明として正しいものはどれか。 | 芽胞を含むすべての微生物を死滅または除去する |
| 消毒 | 消毒の説明として正しいものはどれか。 | 病原微生物を殺すか除去して感染力をなくす |
| 煮沸消毒 | 煮沸消毒の条件として正しいものはどれか。 | 沸騰後2分間以上 |
| 蒸気消毒 | 蒸気消毒の条件として正しいものはどれか。 | 80℃を超える蒸気に10分間以上触れさせる |
| 紫外線消毒 | 紫外線消毒の条件として正しいものはどれか。 | 1平方センチメートル当たり85マイクロワット以上で20分間以上 |
| 化粧品 | 化粧品の説明として正しいものはどれか。 | 清潔・美化などを目的とし、人体への作用が緩和な製品 |
| 医薬部外品 | 医薬部外品の説明として正しいものはどれか。 | 人体への作用が緩和で、一定の効能効果を目的とする製品 |
| 陰イオン界面活性剤 | 主に洗浄力と起泡力を利用する界面活性剤はどれか。 | 陰イオン界面活性剤 |
| 陽イオン界面活性剤 | 毛髪への吸着、帯電防止、柔軟作用に用いられるものはどれか。 | 陽イオン界面活性剤 |
| 非イオン界面活性剤 | 水中でイオンにならず、乳化や可溶化に用いられるものはどれか。 | 非イオン界面活性剤 |
| 両性界面活性剤 | 液性により陽イオン性・陰イオン性の性質を示すものはどれか。 | 両性界面活性剤 |
| 酸化染毛剤 | 酸化染毛剤の特徴として正しいものはどれか。 | 染料中間体とカプラーが酸化反応で発色する |
| 還元剤 | パーマ第1剤で毛髪のシスチン結合を切断する成分はどれか。 | 還元剤 |
| 酸化剤 | パーマ第2剤で切断された結合を再形成させる成分はどれか。 | 酸化剤 |
| パーマ第1剤 | パーマ第1剤の役割はどれか。 | 還元剤によりシスチン結合を切断する |
| パーマ第2剤 | パーマ第2剤の役割はどれか。 | 酸化剤により切断された結合を再形成する |
| 脱色剤 | 脱色剤の主な働きはどれか。 | 毛髪中のメラニン色素を酸化分解する |
| 酸性染毛料 | 酸性染毛料の特徴はどれか。 | 酸性染料で主に毛髪表面や表層部を染める |

合計：24問。選択肢は表示時に並べ替えます。
