# リリースノート

## v2.9.45 過去問PWA更新処理修正版

### 変更ファイル
- `過去問/sw.js`
- `RELEASE_NOTES.md`

### 変更内容
- 過去問用キャッシュ名を `riyo-kakomon-v2-9-2` から `riyo-kakomon-v2-9-45` へ更新しました。
- 過去問ページを「通信優先・通信失敗時のみキャッシュ使用」の方式へ変更しました。
- サービスワーカー有効化時に、`riyo-kakomon-` で始まる古い過去問キャッシュだけを削除する処理を追加しました。
- `skipWaiting()` と `clients.claim()` により、更新を反映しやすくしました。

### 変更理由
- 既存のホーム画面アプリで古い過去問ファイルが優先表示される問題を防ぎ、最新版を反映しやすくするためです。

### 変更していない内容
- トップページ・教材ページのサービスワーカーとキャッシュ
- HTML・CSS・JavaScript本体・問題データ
- 今日の学習モード・問題集モード
- 出題ロジック・学習履歴・達成率・要復習・模試・保存キー
