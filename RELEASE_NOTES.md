# リリースノート

## v2.9.45 過去問PWA更新処理修正版

### 変更ファイル
- `過去問/sw.js`
- `RELEASE_NOTES.md`

### 変更内容
- 過去問用キャッシュ名を `riyo-kakomon-v2-9-2` から `riyo-kakomon-v2-9-45` へ更新しました。
- 過去問ページを「通信優先・通信失敗時のみキャッシュ使用」の方式へ変更しました。
- サービスワーカー有効化時に、`riyo-kakomon-` で始まる古い過去問キャッシュだけを削除する処理を追加しました。
- `skipWaiting()` と `clients.claim()` により、更新を反映しやすくしました。

### 変更理由
- 既存のホーム画面アプリで古い過去問ファイルが優先表示される問題を防ぎ、最新版を反映しやすくするためです。

### 変更していない内容
- トップページ・教材ページのサービスワーカーとキャッシュ
- HTML・CSS・JavaScript本体・問題データ
- 今日の学習モード・問題集モード
- 出題ロジック・学習履歴・達成率・要復習・模試・保存キー

## v2.9.46 サービスワーカー完了待機処理修正版

### 変更ファイル
- `過去問/sw.js`
- `RELEASE_NOTES.md`

### 変更内容
- `self.skipWaiting()` を、キャッシュ登録完了後に実行される `install` イベントの `event.waitUntil()` 内へ移動しました。
- `self.clients.claim()` を、古い過去問キャッシュ削除完了後に実行される `activate` イベントの `event.waitUntil()` 内へ移動しました。

### 変更理由
- サービスワーカーのインストール処理と有効化処理が完了するまで、各イベントの寿命を確実に維持するためです。

### 変更していない内容
- キャッシュ名、キャッシュ対象ファイル、通信優先方式
- 古い過去問キャッシュの削除条件
- HTML、CSS、JavaScript本体、問題データ、保存データ
- トップページ・教材ページのサービスワーカー
