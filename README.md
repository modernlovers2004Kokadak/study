# 理容師国家試験対策 統合版 v2.0.2

GitHub Pages にアップロードするだけで利用できる完成版です。

## 反映内容

- フローティングホームボタンを追加
- ランク表示を小さな色丸付きの「最重要」「重要」「頻出」の縦積みに統一
- 数字を半角数字に統一
- 文字化けしないフォルダ名に整理
- トップページ、教材集、問題集の導線を維持

## 使い方

このZIPを展開し、中身をGitHubリポジトリにアップロードしてください。


## v2.1.0
- フローティングホームボタンをトップ以外に設置
- 苦手ランキングの内部キー表示を防止
- 苦手データがない場合は初期メッセージを表示
- ランク表示を小さなカラー丸と縦3行で統一
- 学習の進捗カードをコンパクト化


## v2.1.0
- 教材の末尾に「この分野の問題を解く」を追加
- 問題解説の末尾に「この内容を教材で確認する」を追加
- トップページ進捗をプログレスバー表示に変更

## v2.4.0
- トップページと教材集の分野別「問題」リンクを1行表示に調整。
- 「この分野の問題を解く」を「問題」に短縮。
- 太字を使わない表示方針を維持。


## v2.4.0
- 今日の学習で正解済み問題の再出題頻度を低減
- 1回正解は7日間、2回連続正解は14日以上、頻出の克服済みは30日間出題を抑制
- 今日の学習21問のうち正解済み問題は最大3問までに制御


## v2.5.0 更新
- 今日の学習で、同じ設問文は同日中に1回だけ出題するよう調整。
- 正解済み・完全克服済み問題の再出題頻度を抑制。
- 未回答・苦手・復習期限到来問題を優先。


## v2.6.0 更新内容
- 設問ID（stemId）とテーマID（themeId）を分離しました。
- 同一問題文を整理し、設問重複が出ない構成にしました。
- 重複を避けた新規問題を追加し、合計650問にしました。
- 4択の選択肢は出題ごとにシャッフルされます。
- 同一設問は同日の「今日の学習」で1回のみ出題されます。
- 完全克服済み問題は低頻度出題になるよう調整しています。
- 忘却曲線に基づく復習を維持しています。
