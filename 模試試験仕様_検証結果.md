# 模試の試験仕様 検証結果

検証日：2026年7月16日

## 出題仕様

模試1回につき55問、次の9区分で固定配分されることを、100回のランダム生成で確認しました。

| 区分 | 問題数 |
|---|---:|
| 関係法規・制度及び運営管理 | 10 |
| 公衆衛生・環境衛生 | 5 |
| 感染症 | 5 |
| 衛生管理技術 | 5 |
| 人体の構造及び機能 | 5 |
| 皮膚科学 | 5 |
| 香粧品化学 | 5 |
| 文化論 | 5 |
| 理容技術理論 | 10 |

各回55問すべてが重複なしで、100回の問題集合は100通りに変化しました。

## 合否・無得点判定

| 条件 | 期待結果 | 結果 |
|---|---|---|
| 55問正解、全区分得点あり | 合格 | 合格 |
| 33問正解、全9区分で1問以上正解 | 合格 | 合格 |
| 32問正解 | 不合格 | 不合格 |
| 33問以上正解だが1区分が0点 | 不合格 | 不合格（無得点区分を表示） |
| 55問未完了 | 不合格 | 不合格（未完了を表示） |

合格条件は「55問中33問以上、かつ9区分すべてで1点以上」です。

## 途中保存・再開

- 模試開始時に問題ID、問題順、終了予定時刻を保存。
- 各解答後に模試専用の解答結果と次の問題位置を保存。
- 17問解答時点で再読み込みした条件で、18問目から再開できることを確認。
- ホームへ戻った後に模試ボタンを押した場合も、未完了の同じ模試を再開。
- 不正な保存データ、55問を満たさない問題列は破棄。

## 履歴

- 通常学習の直近履歴ではなく、模試専用の55件だけから採点するよう修正。
- 合格・不合格回数に加え、受験日、得点、各区分の得点を最大100回保存。
- 結果画面を再描画しても同じ受験を二重計上しないことを確認。

## 制約

アプリ本体と同じJavaScript、DOM更新処理及びlocalStorage互換保存領域による自動実行試験は合格しました。ただし、この作業環境ではローカルURLをクラウドブラウザで開く操作が安全制限により拒否されたため、物理スマートフォン／PCブラウザでのタップ操作を「実機確認済み」とはしていません。公開前にSafari及びChromeで、開始→数問解答→再読み込み→再開→完答を各1回行うことを最終実機確認項目とします。
